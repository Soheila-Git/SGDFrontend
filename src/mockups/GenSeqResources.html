Gene and Sequence Resources

Retrieve Sequence and a list of options for accessing biological information, displays, and sequence analysis tools.

Input type

Gene Name or Identifier

Examples
  • Gene: act1
  • ORF: YHR023W
  • SGDID: S000001855
  • GenBank Locus: YSCHELI
  • GenBank AccNo: L00683
  • GenBank GI: gi:171655

Wildcard: Can enter first few characters followed by *

Multiple entries: Separate multiple entries with commas

Output

Position

The entire chromosome sequence will be displayed if no coordinates are entered.

Enter coordinates in ascending order for the Watson strand and descending order for the Crick strand.
